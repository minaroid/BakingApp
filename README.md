# BakingApp
Baking Application for making delicious recipes.

# Features:
* Share Ingredients of a specific cake.
* Home Screen Widget.
* Watch the Recipes on Video Player.
* Supports Multi-Screen sizes.

# Libraries:
* Android Support Libraries.
* Espresso Test Library.
* ButterKnife.
* Glide.
* Retrofit.
* Gson
* ExoPlayer.
* Dagger2.
* rxJava.
* OkHttp.
